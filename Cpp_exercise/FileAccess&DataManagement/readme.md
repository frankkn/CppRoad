## 最高分數管理程式

這是一個簡單的最高分數的管理程式。假設這是一個遊戲軟體中的一部分功能。會在遊戲啟動的時候，讀入目前遊戲的10個最高分的紀錄。在遊戲過程當中，這些分數會變動。在遊戲結束的時候。會重新把變動後的10個最高分存回檔案。

* 檔案的輸入與輸出
* 集合物件的應用。包含資料的儲存、修改、新增、刪除、排序與搜尋。

* Associate
  * open()
  * is_open()
  * close()

* Input
  * get()
  * eof()
  * getline()
  * >>

* Output
  * put()
  * write()
  * <<

* 如果少讀最後一行，就在txt裡面最後一行文字寫完，再多空一行出來。
